       ```rust
       List(&'a ListArray, DataType),        // child element DataType
       ```

       ```rust
       pub fn prepare_extractor<'a>(
           py: Python<'_>,
           col: &'a dyn Array,
           data_type: &DataType,
           nested_models: Option<&PyObject>,  // None for non-struct, Some(model_cls) for struct fields
       ) -> PyResult<ColumnExtractor<'a>>
       ```

       - `DataType::List(field)` => downcast via `as_list_array(col)`, return `ColumnExtractor::List(arr, field.data_type().clone())`

       - `DataType::Struct(fields)` => The `nested_models` parameter provides the Pydantic model class for this struct. Downcast via `as_struct_array(col)`. Iterate `fields`, for each child field: get child column from struct array via `struct_arr.column(i)`, call `prepare_extractor` recursively for each child (passing nested model class info if child is also a struct). Build `Vec<(Py<PyString>, ColumnExtractor)>` of (interned field name, child extractor). Return `ColumnExtractor::Struct(struct_arr, child_extractors, model_cls)`.

    4. Implement `extract_value` for new variants:

       **List:**
       ```rust
       ColumnExtractor::List(arr, child_dt) => {
           if arr.is_null(row) {
               Ok(py.None())
           } else {
               let child_array = arr.value(row);  // ArrayRef for this row's list
               let len = child_array.len();
               // Create temporary extractor for child elements
               let child_ext = prepare_extractor(py, child_array.as_ref(), child_dt, None)?;
               let mut items: Vec<PyObject> = Vec::with_capacity(len);
               for j in 0..len {
                   items.push(child_ext.extract_value(py, j)?);
               }
               Ok(PyList::new(py, &items)?.into_any().unbind())
           }
       }
       ```

       **LargeList:** Identical to List but uses LargeListArray.

       **Struct:**
       ```rust
       ColumnExtractor::Struct(arr, children, model_cls) => {
           if arr.is_null(row) {
               Ok(py.None())  // null struct -> None for entire nested model (Pitfall 4)
           } else {
               let kwargs = PyDict::new(py);
               for (field_name, extractor) in children.iter() {
                   let value = extractor.extract_value(py, row)?;
                   kwargs.set_item(field_name.bind(py), value)?;
               }
               Ok(model_cls.bind(py).call_method("model_construct", (), Some(&kwargs))?.unbind())
           }
       }
       ```

       Type detection logic:
       ```python
       import typing
       from pydantic import BaseModel

       def _get_nested_model(annotation) -> type[BaseModel] | None:
           """Extract nested BaseModel class from field annotation."""
           # Direct BaseModel subclass
           if isinstance(annotation, type) and issubclass(annotation, BaseModel):
               return annotation
           # Optional[NestedModel] = Union[NestedModel, None]
           origin = typing.get_origin(annotation)
           if origin is typing.Union:
               args = typing.get_args(annotation)
               for arg in args:
                   if isinstance(arg, type) and issubclass(arg, BaseModel):
                       return arg
           return None
       ```
