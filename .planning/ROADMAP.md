## Phases

<details>
<summary>✅ v1.0.0 Core Library (Phases 1–7) — SHIPPED 2026-07-05</summary>

Dict-free, single-step conversion from Arrow buffers to Pydantic model
instances via a Rust core: full Arrow DataType coverage, alias resolution,
a fast (`model_construct`) path and an opt-in validated path, a lazy iterator
API, and type stubs.
